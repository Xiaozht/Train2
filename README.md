# All Aboard! The Magic of Trains 🚂

An interactive, "scrollytelling" introduction to trains designed for kids. This project uses a premium, fluid animation style inspired by Apple's product pages to make learning about trains engaging and visually stunning.

## ✨ Features
- **Scrollytelling**: The narrative unfolds as you scroll down the page.
- **Dynamic Animations**: A train moves across the screen, wheels spin, and smoke rises in sync with user interaction.
- **Responsive Design**: High-contrast colors and bold typography tailored for a younger audience.
- **Parallax Effects**: Clouds and environment elements move at different speeds to create depth.

## 🛠️ Tech Stack
- **HTML5 & CSS3**: For structure and modern styling.
- **GSAP (GreenSock Animation Platform)**: Used for the high-performance animations.
- **ScrollTrigger**: A GSAP plugin that ties animations directly to the scroll position.
- **Google Fonts**: 'Outfit' for a clean, modern look.

## 🚀 How to Run
1. Clone this repository or download the files.
2. Open `index.html` in any modern web browser.
3. Scroll down to start the journey!
